        /== Qu'est ce que c'est un CTF? ===================\
        |                                                  |
        | Il n'y a rien de plus agréable qu'un cadeau que  |
        | l'on découvre après son déballage, non?          |
        |                                                  |
        | Alors, on a voulu rendre le paquet cadeau un peu |
        | plus original...                                 |
        |                                                  |
        |         Voici le "Catch The Flag" (CTF)!         |
        |                                                  |
        | Le CTF est un concours que les Hackers adorent   |
        | faire pour résoudre, apprendre et découvrir des  |
        | accès informatique plus ou moins légaux.         |
        |                                                  |
        | Le but, trouver les 6 solutions (flags) pour     |
        | récupérer tes cadeaux 1337 de noël.              |
        |                                                  |
        | Evidemment, si tu as besoins d'idées pour        |
        | résoudre des énigmes, nous pourrons t'aider.     |
        |                                                  |
        | NB: si tu as des talents de HaCkEr et que tu     |
        | essayes de pirater les pages web, tu trouveras   |
        | des raccourcis....                               |
